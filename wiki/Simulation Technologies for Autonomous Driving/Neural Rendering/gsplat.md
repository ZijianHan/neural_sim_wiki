# gsplat

## Definition
gsplat is an open-source CUDA-accelerated Gaussian rasterization library with Python bindings for building efficient Gaussian-Splatting pipelines.

## Synthesis
### Source-grounded Role
Within this wiki, gsplat is primarily an infrastructure node:
- high-performance rasterization backend,
- reproducible evaluation tooling,
- integration surface for extensions such as 3DGUT.

### 中文版本
